# Phone Management (aliyss-phone)

This folder contains documentation and generated assets for managing the connection and styling of the Android (Termux) device.

## Components

- **`colors.properties`**: Automatically generated by Nix. Matches the desktop's **Oxycarbon** theme.
- **`scripts/sync-phone.sh`**: Helper script to push configuration and assets to the phone.

## How to Sync

To push the latest generated theme and configuration to your phone over Tailscale, run:

```bash
./scripts/sync-phone.sh
```

## Manual Setup on Phone (Termux)

If you are setting up a fresh device:

1. **Install Patched Tailscale (Android 11+ compatible)**:
   ```bash
   curl -fsSL https://raw.githubusercontent.com/bropines/tailscale-termux-cli/main/remote-install.sh | bash
   ```
2. **Start Tailscale Service**:
   ```bash
   tailscaled-start --service=on
   tailscale-cli up
   ```
3. **Install Core Packages**:
   ```bash
   pkg install openssh fish termux-api
   ```
4. **Authorize Desktop**:
   Add the desktop's public key to `~/.ssh/authorized_keys`.
5. **Shell**:
   `chsh -s fish`

## SSH Shortcuts

The phone is accessible via Tailscale at:
- **Hostname**: `aliyss-termux` (or use the IP directly)
- **Port**: `8022`
- **User**: `aliyss`

## Tailscale Commands (Patched version)

- `tailscaled-start`: Starts the daemon or manages the service (`--service=on/off/status`).
- `tailscale-cli`: The command to use instead of raw `tailscale` (it uses the correct socket).
- `tailscaled-log`: View the daemon logs.
- `tailscale-test`: Verify the connection.

## Generated Files

The `colors.properties` file in this directory is a symlink to the Nix store. **Do not edit it directly**. Modify `flake/lib/theme.nix` or `flake/lib/themes/termux.nix` instead.
